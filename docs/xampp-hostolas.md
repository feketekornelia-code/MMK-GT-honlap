# XAMPP hostolás (PHP include verzió)

1. Másold a projektet a `C:\xampp\htdocs\mmk-gt` mappába.
2. Indítsd el az Apache-ot a XAMPP Control Panelben.
3. Böngészőben nyisd meg: `http://localhost/mmk-gt/src/html/index.php`.

## Struktúra

- Fő oldal: `src/html/index.php`
- Bal oldali include: `src/includes/sidebar-left.php`
- Jobb oldali include: `src/includes/sidebar-right.php`
- Stílusok: `src/css/styles.css`

## Megjegyzés

- A CSS és kép útvonalak a fenti URL-struktúrához vannak igazítva.
- Ha másik mappanévvel teszed `htdocs` alá, csak az URL első része változik.
